Welcome

This project showcases data analysis and visualization using Python. While named for a COVID-19 tracker, it currently analyzes the Iris dataset to demonstrate data processing techniques applicable to global health data. Explore the visualizations and insights below!

Objectives

  • Load and explore a dataset using pandas.
  • Perform basic statistical analysis.
  • Create visualizations to uncover patterns.
  • Present findings in an accessible, web-based format.

Visualizations

Line Chart: Mean Feature Values

Shows trends in mean feature values across Iris species.

Bar Chart: Average Petal Length

Compares average petal length by species.

Histogram: Sepal Length Distribution

Displays the distribution of sepal length.

Scatter Plot: Sepal vs. Petal Length

Visualizes the relationship between sepal and petal length, colored by species.

Tools & Setup

Tools and Libraries

  • Python: Programming language.
  • pandas: Data manipulation and analysis.
  • matplotlib: Plotting and visualization.
  • seaborn: Enhanced visualization styling.
  • scikit-learn: Iris dataset source.
  • Jupyter Notebook: Interactive coding environment.

How to Run

  1. Clone the repository: git clone https://github.com/your-username/your-repo.git
  2. Install dependencies: pip install pandas matplotlib seaborn scikit-learn
  3. Open iris_analysis.ipynb in Jupyter Notebook.
  4. Run all cells to reproduce the analysis and visualizations.

View the notebook on GitHub: iris_analysis.ipynb

Insights

  • Species Separation: Setosa is distinctly separable due to its smaller petal measurements, ideal for classification tasks.
  • Feature Differences: Virginica has the largest sepal and petal measurements, followed by versicolor.
  • Visualization Clarity: The scatter plot effectively shows species clusters, highlighting feature correlations.
  • Applicability: Techniques used here (e.g., grouping, plotting) are transferable to global health data like COVID-19 case tracking.

About

This project was developed to demonstrate data analysis and visualization skills. While the Iris dataset was used for simplicity, the methods are applicable to real-world datasets like COVID-19 case data. Reflections:

  • Learned the power of pandas for data manipulation and matplotlib for visualization.
  • Understood the importance of clear, labeled plots for communicating insights.
  • Recognized the need for robust error handling in data pipelines.
  • Future work could integrate real COVID-19 data from sources like WHO or Johns Hopkins.
